# PhiPluginAdv
一些bot新功能测试

### 随机课题

依赖于Phi-Plugin的定数表文件，没有会报错。

#### 命令参数详解

#随机课题 [定数] [难度] [平均]

例：

#随机课题 30 HD → HD难度总和30

#随机课题 45+ AT 平均 → AT难度总和≥45，三首定数差≤1

#### 附注

随机课题上述参数均不是必要传入条件，同时/或者#均可。
